易科（Yix） 是一种全新的形式语言，用于描述任何一种存在形式的生命周期，并提供相应
的显示/诊断/预测等功能。

易科引擎是基于 Vue 的是一种具体的实现，用来运行使用易科编写的应用程序，展示域空
间的生命周期。
